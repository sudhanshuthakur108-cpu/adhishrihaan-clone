# Adhishrihaan Foundation Clone 🌍

A responsive homepage clone of the Adhishrihaan Foundation website built using **HTML, CSS, JavaScript, and Bootstrap 5**.

## 🚀 Live Demo

🔗 https://adhishrhaan.netlify.app/

## 📂 GitHub Repository

🔗 https://github.com/sudhanshuthakur108-cpu/adhishrihaan-clone

---

## 📌 Project Overview

This project is a responsive frontend clone created as part of an assignment to demonstrate:

* Responsive web design principles
* Bootstrap grid system implementation
* Clean UI structure
* Reusable components
* Mobile-first design approach

The website replicates the overall structure, layout, and sections of the original Adhishrihaan Foundation website.

---

## ✨ Features

* Fully Responsive Design
* Bootstrap 5 Grid Layout
* Hero Section
* About Section
* Services Section
* Statistics Section
* Clean Footer Design
* Smooth Card Hover Effects
* Mobile-Friendly Layout
